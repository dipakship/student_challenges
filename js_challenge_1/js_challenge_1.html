JavaScript Challenge 1- Cooking School

Instructions:

  1. Using the JavaScript page, add a second prompt to ask the user their last name.
  2. Change the alert to greet the person by their first and last name.
  3. Add a picture of a person cooking to the bottom of the page.
    • Style using CSS to have the picture fit the page.
  4. Add a button for the user to apply to the cooking school.
    • Syle using CSS to add colors that fit the current theme of the page.

  5. Ask the user how old they are.
    • Set the rule so users under age 14 are alerted that they cannot apply.
    • Users 14 and older should be given a welcome alert.
    • Hint: Use an if/else statement to help you
